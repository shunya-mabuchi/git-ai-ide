# トリアージラベル

`docs/agents/issue-tracker.md` と GitHub Issues で使うラベルです。

## 状態

- `要整理`: まだ内容を確認していない
- `情報不足`: 仕様や判断材料が足りない
- `実装可能`: Codex が実装できる粒度まで整理済み
- `人間判断待ち`: ユーザーやプロダクト判断が必要
- `作業中`: 実装中
- `完了`: 実装と検証が完了
- `対応しない`: 意図的に対応しない
